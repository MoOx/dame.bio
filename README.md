## Contribute

### Installation

```console
git clone https://github.com/MoOx/dame.bio
cd dame.bio
yarn
```

### Start

#### Web

##### If your editor support [LSP](https://microsoft.github.io/language-server-protocol/specification#initialize)

```console
yarn start:web:js
```

##### Otherwise

```console
yarn start:web
```
